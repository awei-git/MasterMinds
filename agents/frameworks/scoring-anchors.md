# 评分锚定库

知音（reader）评分时的参照物。每本书按7个维度独立打分，基于创作者的真实阅读体验校准。

**使用方法**：评分时对照这些锚点。比如你给"语言质感"打9分，问自己：这段文字的语言质感真的接近《边城》或《黄金时代》吗？如果不是，降分。

> 分数反映创作者的真实阅读体验，不是文学史地位。一本"伟大"的书如果读不下去，翻页欲望就是低的。

---

## 维度说明

| 维度 | 评的是什么 |
|------|-----------|
| 主题贯穿 | 主题是活在故事里的，还是贴上去的标签？ |
| 故事推进 | 每一章都在把故事往前推吗？有没有废章？ |
| 人物可信 | 角色做的事、说的话，像真人吗？ |
| 情感冲击 | 读完有没有被打到？情绪是earned还是强塞的？ |
| 语言质感 | 句子有呼吸吗？有没有想抄下来的句子？ |
| 节奏控制 | 快慢交替合理吗？有没有该快的拖了、该慢的赶了？ |
| 翻页欲望 | 凌晨3点还想往下读吗？ |

## 校准规则

1. **翻译体天花板**：翻译作品的语言质感通常不超过8分，除非翻译后仍有独特语言能量（如《百年孤独》《灵山》的译本）
2. **读不下去 = 翻页低分**：无论文学史地位多高，个人读不下去就是低翻页分
3. **语言好 ≠ 吸引人**：《边城》语言10分但翻页4分。好句子不等于好故事
4. **人物不可信会拖垮一切**：《那不勒斯》人物别扭导致情感冲击也打不上去
5. **一口气读完 ≥ 9分翻页**：不管是悬念驱动还是情感驱动，停不下来就是停不下来
