# Compression

**原则**：好的prose不是写出来的，是删出来的。每一个字都要挣得自己的位置。

---

## 何时启动

- 初稿完成，准备修改
- 一句话里形容词≥3个
- 同一段出现两句话表达同一个意思
- 段落开头在铺垫而不是在行动
- 删掉某段的第一句，段落意思不变
- 读出声时某处让你换气或想跳过

---

## Elmore Leonard 核心规则

1. **不用"说"以外的引语动词。** "喃喃道""低吟""沉声道"——全删，回到"说"或直接省略。
2. **不用副词修饰"说"。** "愤怒地说"→删副词，写动作。"他把杯子摔在桌上。'滚。'"
3. **不写天气，除非角色在乎天气。** 开篇下雨不是氛围，是偷懒。
4. **不写读者会跳过的段落。** 如果你自己重读时眼睛滑过去了，读者也会。

**判断标准**：如果读起来像"写作"，重写。

---

## 压缩的层次（从大到小）

**先砍结构，再磨句子，最后挑字。** 反过来做，你会花一小时打磨一个段落，然后整段删掉。

### 段落级

**进场晚**：删铺垫，从事情已经发生的地方开始。
- ✗ 那天早上天气不错。他吃了早饭，出了门，开车到了公司。会议在十点开始。他走进会议室，发现只有李明一个人。
- ✓ 会议室里只有李明一个人。

**离场早**：场景的意义传达完了就走。不写角色怎么离开房间。

**删重复情绪**：同一种感受用两种方式表达了（一次动作，一次内心），删内心那句。

### 句子级

**删解释句**：动作已经传达了信息，不要再用陈述句复述。
- ✗ 他摔门而去。他很愤怒。
- ✓ 他摔门而去。

**删过渡句**：从A到B，直接写B。读者会自己填路径。
- ✗ 于是他决定去找她。他穿过街道，来到她家门前。
- ✓ 他站在她家门前。

**删总结句**：段落末尾复述刚发生的事——读者三秒前刚读过，不需要。

### 词语级

**删形容词**：一句话里超过两个形容词，删一个。
- ✗ 他走进阴暗潮湿狭窄的巷子
- ✓ 他走进潮湿的窄巷

**删副词**：副词在补救动词的无力——换一个更准确的动词。
- ✗ 她快速地走过走廊
- ✓ 她穿过走廊
- ✓✓ 她掠过走廊

**删程度词**："很/非常/十分"——直接删，不替换。
- ✗ 他非常愤怒
- ✓ 他愤怒
- ✓✓ （写他做了什么）

**删启动词**："开始/继续/不由得"——几乎总是多余的。
- ✗ 她开始哭泣
- ✓ 她哭了

---

## 压缩检验（按顺序做）

1. **读出声**：读起来有阻力的地方标记。阻力＝多余。
2. **逐句遮盖**：遮住一句话，上下文还通吗？通→该删。
3. **还原检查**：读一遍删后的版本。如果某处节奏断了、声音变了、情绪塌了——把那个词或句子放回去。具体判断：
   - **节奏停顿**：一个"多余"的词可能是呼吸点。删了之后两句撞在一起→放回去。
   - **声音标记**：某个词是这个叙述者的语言习惯，不是冗余。"他妈的"可能是废话，也可能是人物。
   - **情感留白**：一句看似空洞的话在承载情绪。"外面下雨了。"——如果删了之后段落只剩事件没有温度，它不是废话。
4. **终检**：再读一遍。确认每个字都在工作。

**过度压缩的样子**：
- ✗ 会议室。李明。沉默。她进来。→ 这不是密度，是电报。节奏死了。
- ✓ 会议室里只有李明一个人。她推门进来时，他没有抬头。→ 密度高，但还在呼吸。

---

**原则**：压缩是为了密度，不是为了短。一段话可以很长，但每个字都不多余。
